Lesson 1: Agentic RAG

Summary:

- Understand what RAG is and how it works
- Build a search engine over a real FAQ dataset
- Write a prompt that combines the user's question with search results
- Wire it all together into a working RAG pipeline
- Split ingestion and query into separate processes

We also learn about make the pipeline agentic. The LLM decides when and what to search, instead of running the same fixed flow every time.

In the 01-agentic-rag homework includes how we build a FAQ agent for our course from FAQ data prepared.
